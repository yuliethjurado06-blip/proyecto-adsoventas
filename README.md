# Proyecto Formativo

Aplicación web desarrollada como parte de una actividad académica. El proyecto implementa una arquitectura MVC y funcionalidades de gestión de información mediante una interfaz web.

## Tecnologías Utilizadas

- HTML5
- CSS3
- Bootstrap
- PHP
- MySQL
- Git

## Instalación

1. Instalar el repositorio.
2. Configurar la base de datos.
3. Ejecutar el proyecto en un servidor local como XAMPP o Laragon.
